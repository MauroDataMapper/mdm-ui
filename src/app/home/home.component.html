Benefits of using the Mauro Data Mapper

Automate

It is safer to produce a new artefact by copying parts of a model than by copying parts of the existing implementation.

Collaborate

It is easier to compare data models, written in a single modelling language, than to compare artefacts implemented using a range of different technologies.

Generate

Automatically import data models from XML Schema, relational databases such as PostgreSQL, MS SQL Server and OracleDB
